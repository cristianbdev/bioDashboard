# Bio Dashboard

Dashboard para visualizar resultados de bioseguridad en acuicultura consumiendo KoboToolbox.

## Package manager oficial

Este proyecto usa **Bun** como gestor de paquetes y runner de scripts.

- Instalar dependencias: `bun install`
- Desarrollo: `bun run dev`
- Lint: `bun run lint`
- Build: `bun run build`
- Producción: `bun run start`

## Entorno

Configura un archivo `.env.local` con las variables necesarias (Kobo + Clerk).

## Notas

- Stack principal: Next.js + React + TypeScript + Tailwind CSS v4 + Bun.
- No se usan otros gestores de paquetes en este proyecto.
